# AI Chatbot: Engineering Assistant

## Overview

An AI-powered engineering assistant built with Next.js and DeepSeek API, featuring streaming responses and syntax-highlighted code blocks for improved developer readability.

## Key Features

- **Streaming Responses:** Real-time streaming of AI responses for reduced perceived latency
- **Syntax Highlighting:** Code blocks with proper syntax highlighting for various languages
- **Responsive Design:** Optimized for both desktop and mobile experiences
- **Smooth Animations:** Framer Motion powered text-generation animations

## Tech Stack

- Next.js for the frontend framework
- React for UI components
- TypeScript for type safety
- Tailwind CSS for styling
- Framer Motion for animations
- DeepSeek API for AI responses

## Development Highlights

- Built a modular React UI with Tailwind CSS and Framer Motion
- Enabled rapid iteration and smooth text-generation animations
- Implemented streaming API responses for better UX
- Created a polished, production-ready interface
